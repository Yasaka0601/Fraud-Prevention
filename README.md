## サービス概要
このアプリケーションは、家族間で特殊詐欺に関するクイズを出題・回答しながら、防犯意識を高めることができる学習型アプリです。
離れて暮らす家族に出題したり、出題を受けたりしながら、詐欺の手口を知り、正解すると加算されるポイントで競い合うことで、日常的に詐欺対策を意識づけることを目的としています。

クイズの結果は家族間で共有され、どの程度の知識が身についているかを可視化します。
ランキングや称号機能を通して楽しく学べる環境を提供します。
最終的には、家族ぐるみで「詐欺に強い家庭」を育てることを目指します。
目指せ、特殊詐欺被害ゼロ！
## このサービスへの思い・作りたい理由
一人暮らしをしている私のスマートフォンが鳴り、電話に出た。
電話の主は、離れて暮らしている両親だった。
両親からの電話は、久しぶりだった。
両親は「警察から電話があった」とのことで、内容は「母の口座が使い込まれている」というもので、加えて、資産状況などを質問されたとの事だった。
両親は、特殊詐欺の防犯意識は高いと思っていたが、どうやらこの電話が、詐欺なのか分からない様子であった。
私は、「その電話は詐欺である」と告げた上、不審な電話には対応しないようにと伝えた。
ここで、ふと、私は思った。
「何を根拠に詐欺の電話と断定したのか」「不審な電話とは具体的には何を指すのか」を両親に説明することが出来なかったのだ。
私は、さらに考えた。
「自分が特殊詐欺について詳しくなれば、両親に具体的に安全策を提示できる」
「両親がどれくらい特殊詐欺に対する知識があるのかを把握することが出来れば、離れて暮らしていても安心できる」
「家族全員が詐欺への理解を深める」ための仕組みの必要性を痛感したのである。
さらに、警察庁が公表した最新の統計では、特殊詐欺の被害は深刻化している。
•	認知件数：15,583件（前年同期比 +48.4%）
•	被害総額：722.1億円（前年同期比 +153.9%）
•	うち「警察官を装った詐欺」が全体の35.9%・被害額の65.2%を占める。
過去最悪だった前年をすでに上回るペースで増加しており、もはや漠然とした知識では詐欺を防ぐことはできない。
被害を食い止めるには、特殊詐欺を未然に防ぐシステムを構築しなければならない。
こうした背景の中、「知識の共有」と「家族の見守り」を両立できる仕組みとして、本サービスを構想したものである。
## ターゲットユーザー層について
•　MVP段階
　　離れて暮らす高齢の両親・祖父母と、その家族（子・孫世代）
•　本リリース段階
　　対象を拡大し、若年層の詐欺被害（SNS詐欺・ロマンス詐欺・マッチングアプリ詐欺など）にも対応。
　　年代ごとの詐欺リスクに応じたクイズを提供することで、全年齢層の防犯意識を育てます。
## サービスの利用イメージ
１　まず、ホストユーザー（子世代）が会員登録をし、家族の「ルーム」を作成します。
２　招待コード、URLを通じて、子ユーザー（高齢者の親や、祖父母）を「ルーム」に招待します。
３　「ルーム」に入った後、クイズを解いていきましょう。クイズを通じ、詐欺知識を学習します。
４　家族全員の成績をルーム内で共有、閲覧可能なので、成績が悪ければ危機感を感じ、成績が良ければ、安心感を得る　ことが出来ます。
５　学習してほしい内容は、出題機能を使用することで出題可能です。（本リリース機能）
　　相手のユーザーは出題を受けると、通知され、クイズに挑戦することになります。
　　出題したユーザーは、回答結果を通知で受け取ることが出来ます。
６　正解数に応じてポイントや称号を獲得し、詐欺知識の習得状況を可視化することが出来ます。
７　ランキングで競い合いながら、ゲーム感覚で学ぶことが可能です。（本リリース機能）
８　高齢者が使用することが前提となるので、説明書、ガイドが不要な直感で操作出来るUI/UX設計となっております。
## ユーザーの獲得について

•　RUNTEQ内コミュニティでの初期導入を想定。
## サービスの差別化ポイント・推しポイント
特殊詐欺のクイズを出題しているWebサイトは、警察のホームページをはじめとし、多数存在しています。
しかし、本アプリは３点の差別化ポイントがあります。
１点目は「詐欺知識の習得状況を可視化すること」です。
ポイント制、ランキング制を導入することによって、どれくらい知識が身についているのかを測れるという点です。
２点目は「クイズを家族に出題できる機能」です。（本リリース機能）
解いてほしいクイズ、知っておいてほしい内容についてピックアップし、家族ルーム内のユーザーに出題することが出来ます。
３点目は「情報提供およびクイズ投稿機能」です。（本リリース機能）
アプリ利用者の皆さんで、クイズを作成し、アプリ制作者による審査に通過すれば、クイズを投稿することが出来ます。
また、いただいた情報を基に、アプリ制作者がクイズを作成します。
詐欺の手口は、日々、巧妙化、複雑化していることから、皆様の実体験や、得た最新情報が貴重な情報資源となります。
## 機能候補
MVPリリーズ機能
・ホストユーザー登録機能
　Googleログイン機能
　メールアドレスによる認証
・子ユーザー登録機能
　高齢者向けに、ホストユーザーが作成した子ユーザーによる簡単ログイン。
　「家族ルーム名」と「合言葉」のみでログイン可能にする。
　ルームに参加すると、続柄を選択して、名前を入力し、子ユーザーを登録する。
・ルーム招待機能
　メール,SMSで招待文を送る（URL）による招待。
　本文にはURLと、家族ルーム名、合言葉が記述されており、家族ルームの参加方法が丁寧に記述されている。
・家族ルーム機能
　各ユーザーの採点履歴の閲覧および共有。　
・クイズ機能
　不審電話コース（解説付き）
・ポイント機能
　クイズに正解するとポイントが加算。
・高齢者に配慮したUI/UX設計。

本リリース機能
・高齢者が使用するのを想定した、アプリ使用までの導線簡略化
　１　LINE招待機能
　２　スマートフォンでの使用を想定したレスポンシブデザイン
　３　ガワネイティブ化（余裕があれば）
・ランキング機能
　家族ルーム内のランキングと、家族ルーム単位でのランキングの両方を閲覧することが出来る。
　他の家族とポイントを競い合いましょう。
・X 投稿機能（クイズの成績、称号）
・クイズ再挑戦機能
　間違えた問題をまとめたコース
・クイズ出題機能
　家族に解いてほしい問題について、出題する。
　出題を受けるとLINE通知がある。
　クイズを終えると、採点結果が出題者側にLINE通知される。
・クイズコースの追加
　不審メールコース
　偽サイトコース
　激ムズコース
・情報投稿
　「このような不審電話が流行っている」などの断片的な情報を募集。
　それもを元に新たなクイズを作成。
・クイズ投稿機能
　ユーザーがクイズを投稿する。
　クイズが実装されたら、クイズに作成者名が併記される。
　クイズ職人の称号も実装
・クイズ正解率機能
